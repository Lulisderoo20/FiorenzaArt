# FiorenzaArt

Static portfolio website for Fiorenza Art.

## Tech

- HTML
- CSS
- Vanilla JavaScript

## Run locally

Open `index.html` directly, or serve with a local server:

```powershell
python -m http.server 5500
```

Then open:

- http://127.0.0.1:5500/

## Current MVP scope

- Hero section
- About section
- Art portfolio gallery
- Shop intent section (commissions/collaboration)
- Contact section
- Mobile menu toggle
